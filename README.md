# 🚀 AWS Static Website Hosting with CI/CD

> A complete AWS DevOps project demonstrating secure static website hosting using Amazon S3, Amazon CloudFront (Origin Access Control), AWS CodePipeline, IAM, and Amazon CloudWatch.

---

## 📌 Project Overview

This project demonstrates how to deploy and manage a secure static website on AWS using modern DevOps practices.

The website is hosted on Amazon S3, delivered globally using Amazon CloudFront, automatically deployed through AWS CodePipeline integrated with GitHub, secured with IAM and Origin Access Control (OAC), and monitored using Amazon CloudWatch.

---

## ✨ Features

- Static Website Hosting using Amazon S3
- Secure Content Delivery with Amazon CloudFront
- Origin Access Control (OAC)
- Continuous Deployment using AWS CodePipeline
- GitHub Integration
- IAM Users, Groups & Policies
- CloudWatch Dashboard
- CloudWatch Alarm with SNS Notification
- Responsive HTML, CSS & JavaScript Website
